# Headless Shopify with React

Used Shopify platform as Headless CMS with React App. 


## Tech Stack

**Client:** React, TailwindCSS, Shopify Buy SDK

**Server:** No Server Required


## Features

- Use Shopify as Headless Backend
- Stores Data locally 
- Cross platform (PWA)
- Dark/Light Mode (Upcoming)
